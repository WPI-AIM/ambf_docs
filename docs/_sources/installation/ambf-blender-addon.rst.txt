Installing AMBF Blender Addon
=============================

Bender addon subsect
--------------------

**AMBF Blender Addon**  - package allowing for creating and interacting with robot models described in AMBF YAML format.
Please clone it from its [GitHub repository](https://github.com/WPI-AIM/ambf_addon "AMBF Blender Addon repository").
